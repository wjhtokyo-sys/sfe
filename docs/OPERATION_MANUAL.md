# SFE 操作手册（非IT版）

## 一、怎么启动
1. 打开 VS Code 终端，进入 `C:\sfe-system`
2. 运行后端：
   `\.venv\Scripts\python -m uvicorn main:app --reload`
3. 新开终端运行前端：
   `cd frontend` 后执行 `npm run dev`
4. 浏览器打开：`http://localhost:5173`

## 二、登录入口
- 客户登录入口页：首页上半部分
- 管理员登录入口页：首页下半部分

## 三、模拟账号
### 客户（3组）
- cust1 / 123456
- cust2 / 123456
- cust3 / 123456

### 管理员
- admin1 / admin123
- super1 / admin123（超级管理员）

## 四、菜单说明
### 客户登录后
- 商品信息查询
- 订单管理
- 账单管理
- 历史账单

### 管理员登录后
- 客户管理
- FIFO管理
- 订单管理
- 账单管理
- 历史账单管理

## 五、演示一条完整流程
1. 管理员登录
2. 查看客户、商品
3. 录入订单
4. 录入到货（FIFO批次）
5. 执行FIFO分配
6. 生成账单
7. 推进状态：付款->收款确认->发货->收货->归档
8. 客户登录可查看自己的订单/账单

## 六、按钮效果
所有按钮点击时都会有轻微“回弹”视觉效果。

## 七、常见问题
- 登录失败：确认账号密码是否与上方一致
- 页面空白：确认后端和前端都在运行
- 看不到数据：先运行 `python seed.py` 写入模拟数据
